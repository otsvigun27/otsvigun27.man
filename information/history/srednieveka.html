  • Сортування бульбашкою / Bubble sort

    • Будемо йти по масиву зліва направо. Якщо поточний елемент повинна перевищувати, міняємо їх місцями. Робимо так, поки маса не буде відсортований. Зауважимо, що після першої ітерації найбільший елемент буде знаходитися в кінці масиву, на правильному місці. Після двох ітерацій на правильному місці стоятимуть два найбільших елемента, і так далі. Очевидно, не більше ніж після n ітерацій масив буде відсортований. Таким чином, асимптотика в гіршому і середньому випадку - O (n2), в кращому випадку - O (n).

    void bubblesort(int* l, int* r) {

    int sz = r - l;

    if (sz <= 1) return;

    bool b = true;

    while (b) {

    b = false;

    for (int* i = l; i + 1 < r; i++) {

    if (*i > *(i + 1)) {

    swap(*i, *(i + 1));

    b = true;

    }

    }

    r--;

    }

    }
